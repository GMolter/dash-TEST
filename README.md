# Olio

This repository contains two related applications:

- `OlioWorkstation/` — the existing Vite, React, TypeScript, Supabase, and Vercel web application.
- `OlioLauncher/` — the Windows AutoHotkey v2 launcher application.

## Development

Run Olio Workstation commands from the `OlioWorkstation` directory.

For Vercel deployments, set the project's Root Directory to `OlioWorkstation` so its
`vercel.json`, `package.json`, `api`, and build output are resolved together.

Development of Olio Launcher is governed by `OlioLauncher/roadmap.md`.
